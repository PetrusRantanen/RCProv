# RCProvisioning

A provisioning system for the R-Cooker ovens Compute Module computers.

## Getting Started

### Dependencies

* Raspberry Pi computer with two Ehternet ports
* Raspberry Pi OS (Raspbian) 64-bit Lite

### Installing

**This must be gone to fresh installation of Raspberry Pi OS**

* Clone repository to computer
````
git clone --recurse-submodules --shallow-submodules --depth=1 https://github.com/rmenuoy/rcprovisioning.git
````

* Change directory to rcprovisioning folder and execute install as root
```
cd rcprovisioning
sudo ./install.sh
```

### Executing program

The programs run automatically background.
